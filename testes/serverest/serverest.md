# SERVEREST

https://serverest.dev/


## LOGIN

Realizar o Login para obter o token para realizar algumas requisições

## Usuários

- (POST) Cadastrar 
- (GET) Listar todos
- (GET) Listar por {_ID}
- (PUT) Editar por {_ID}
- (PATCH) Editar por {_ID} - Não implementado. Dependendo o tamanho do JSON pode ser mais eficiente do que o PUT
- (DELETE) Deletar por {_ID}
